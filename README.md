Веб-приложение для учета просмотренных фильмов с возможностью добавления, удаления, фильтрации и сохранения данных на сервере.
Реализовано на чистом JavaScript с использованием Fetch API для взаимодействия с бэкендом.

Функциональность:
Добавление фильмов — форма с полями: название, жанр, год выпуска, отметка о просмотре
Отображение списка — таблица со всеми добавленными фильмами
Фильтрация — поиск по названию, жанру, году и статусу просмотра (все/просмотрено/не просмотрено)
Удаление — кнопка удаления для каждого фильма
Удаление всех — единая кнопка для очистки всего списка
Серверное хранение — все данные сохраняются на удаленном сервере через REST API

Технологии:
JavaScript (ES6+) — асинхронные запросы (async/await), работа с DOM, обработка событий
Fetch API — взаимодействие с сервером (POST, DELETE, GET с параметрами)
HTML5 / CSS3 — семантическая верстка, формы, таблицы, адаптивные элементы
REST API — интеграция с внешним сервером для хранения фильмов
Git — контроль версий
