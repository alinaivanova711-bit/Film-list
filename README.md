Веб-приложение для учета просмотренных фильмов с возможностью добавления, удаления, фильтрации и сохранения данных на сервере.
Реализовано на чистом JavaScript с использованием Fetch API для взаимодействия с бэкендом.

Функциональность:
1)Добавление фильмов — форма с полями: название, жанр, год выпуска, отметка о просмотре;
2)Отображение списка — таблица со всеми добавленными фильмами;
3)Фильтрация — поиск по названию, жанру, году и статусу просмотра (все/просмотрено/не просмотрено);
4)Удаление — кнопка удаления для каждого фильма;
5)Удаление всех — единая кнопка для очистки всего списка;
6)Серверное хранение — все данные сохраняются на удаленном сервере через REST API;

Технологии:
1)JavaScript (ES6+) — асинхронные запросы (async/await), работа с DOM, обработка событий;
2)Fetch API — взаимодействие с сервером (POST, DELETE, GET с параметрами);
3)HTML5 / CSS3 — семантическая верстка, формы, таблицы, адаптивные элементы;
4)REST API — интеграция с внешним сервером для хранения фильмов;
5)Git — контроль версий;
